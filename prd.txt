1. Product Overview
The Event Dashboard is a comprehensive equestrian event management platform designed to
streamline the end-to-end lifecycle of equestrian competitions — from event creation and club
registration to rider management, horse tracking, stable booking, and financial reporting.

1.1 Objectives
•
•
•
•
•

Centralize event management for administrators, clubs, and riders on a single platform.
Automate registration workflows, payment tracking, and financial reconciliation.
Provide real-time analytics and reporting across all modules.
Enforce role-based access to ensure data integrity and operational security.
Support audit logging to maintain a full trail of administrative actions.

13. Non-Functional Requirements
Requirement

Detail

Security

HTTPS enforced. Data encrypted at rest and in transit. Session
expiry controls.

Performance

Page load under 2 seconds for list views with 1,000+ records.

Accessibility

WCAG 2.1 AA compliant UI components.

Scalability

Horizontally scalable backend to support peak event registration
loads.

File Uploads

Max 2MB per file. Supported formats: PDF, JPG, PNG.

Export

CSV and Excel export available across all major list views.

Audit Logging

All admin actions logged with user identity and timestamp.

GST Compliance

Financial module must correctly compute CGST, SGST, IGST per
transaction.
